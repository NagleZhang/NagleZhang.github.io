# 问题汇总

1. 代码追溯困难，在代码向上追溯的过程当中，如果是遇到 configuration 的管理，如果没有在当前代码中定义，则很难梳理当前代码的逻辑。
2. cli 好用，但是仅限于在交互环境中，如果是一些项目当中，代码里面使用 cli， 会造成困扰。而且 cli 的颗粒度实在是太大。用语言调用api 的方式，可以更好的进行复用。
